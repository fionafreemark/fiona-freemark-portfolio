The Daily Tarot

A daily tarot app that allows the user the pull a randomized card from a standard 77-card deck. A local image is then matched to the selected card from the API and displayed on screen with it's meaning and description. The user has the option to save their response in a Firebase database. Saved cards are displayed in the "Saved Responses" section.
